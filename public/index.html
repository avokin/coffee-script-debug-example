How to debug CoffeeScript code of this project

1. Install JetBrains Chrome Extension

There is jb.crx file in the root directory of this application, open Chrome's settings, select "Extension" pane and drag and drop jb.crx from file explorer to chrome. Then turn on the checkbox "Allow access to file URLs" and restart the extension (by disabling it and enabling it back).

2. Connect to Chrome

From Run Configuration list select "CoffeeScript Debug" and start it. RubyMine connects to Chrome, opens page to debug and Chrome should confirm it with yellow bar with text "JetBrains Chrome Extension is debugging this tab." on the top of the page.

3. Set breakpoint and fire the event

Open file debugging_example.js.coffee put breakpoint in the body of method "onButtonClick" and press the big green button on the page. Execution of CoffeeScript code should stop where breakpoint is located.
